The initial total area is

72+80+130=28272 + 80 + 130 = 28272+80+130=282

Each overlap removes shading from each shape involved in the overlap, so the shading is reduced by

The number of shapes overlapping x size over lap

If we calculate the areas that overlap we get

(5 x 4) + (2 x 3) + (6 x 1) = 32

Each overlap includes at least two shapes (32 x 2) = 64

However the central overlap (2 x 1) includes all 3 shapes, so we lose an additional 2 meaning total loss = 66.

282 -66 = 216
