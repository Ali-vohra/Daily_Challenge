Fill the Cross

The goal of a 3×33\times 33×3 Calcdoku puzzle is to fill the squares with a single digit each, with the following rules:

  1. Each of the digits 111 through 333 is used exactly once in each row and column. (There is no region rule as in Sudoku.)

  2. Any marked region indicates a number and an operation (+,−,×,(+,-,\times,(+,−,×, or ÷).\div).÷). The operation applies to digits in the region, and when applied to the digits in some order, they result in the given number. In the example below, the region marked “ 7+"“\, 7+"“7+" must be filled with three digits selected from 111 through 333 that have a sum of 7.7.7. Digits can be repeated in a marked region, but not in a row or column.

    from itertools import permutations

numbers = list(permutations(range(1, 6)))
grid = [(0, 0, 0, 0, 0) for i in range(5)]


def columns_different(row_number):
    for i in range(0, row_number):
        for j in range(5):
            if grid[row_number][j] == grid[i][j]:
                return False
    return True


def calcDokuSolve(n):
    if n < 5:
        for number in numbers:
            grid[n] = number
            if not columns_different(n):
                continue
            if n == 1:
                if not grid[0][1] * grid[0][2] * grid[1][0] * grid[1][1] == 144:
                    continue
            if n == 2:
                if not (grid[0][3] * grid[1][3] * grid[1][4] * grid[2][4]) == 50:
                    continue
            if n == 3:
                if not (grid[1][2] + grid[2][2] + grid[2][1] + grid[2][3] + grid[3][2] == 15):
                    continue
            if n == 4:
                if not (grid[2][0] + grid[3][0] + grid[3][1] + 4 + grid[4][1] == 13 and
                        grid[3][3] + grid[3][4] + grid[4][2] + grid[4][3] == 14):
                    continue
            calcDokuSolve(n + 1)
    else:
        print(grid)
        print(sorted([grid[1][2], grid[2][2], grid[2][1], grid[2][3], grid[3][2]]))


calcDokuSolve(0)

    Let's address the cells as on the chessboard, with lower left being A1. Since the only way to factor 144 is 4×4×3×4, and there is already a 4 in the leftmost column, so A4=3, B4=4, B5=3, C5=4: Now, the only way to factor 50 is 5×5×2×1×1, and there is already 1 in E5, so the only place we can put the other 1 into is D4. The two 5s must go into columns D and E, and there is only one space left in column D it's D5: Now, what are the options to put in 15+ region?

    5+5+3+1+1 remember there is a 5 in either E3 or E4; if it's in E3, then we can't place two 5s inside the 15+ area; so, it must be in E4, and so, E3 is 2. Then, the 5s inside 15+ area can only be in B3 and C2: Now we don't have space for two 1s

    5+4+4+1+1 can't have two 4s because columns B and C already have them, only one place is available for a 4 in D3

    5+4+3+2+1 has no repeating digits

    5+3+3+3+1 can't have three repeat digits: all columns need to be used, and that means, putting a 3 in both B3 and D3, which is the same row

    * 5+3+3+2+2* since there is already a 3 in column B and in row 4, we can only place the two 3s in C2 and D3: But recall the 50x area has a 2 in either E3 or E4, which prohibits us from placing two 2s in the rows 3 and 4

    4+4+4+2+1 can't have three repeat digits, see 5+3+3+3+1

    4+4+3+3+1 columns B and C already have 4s, the only remaining space is D3, and we can't place two 4s there

    4+4+3+2+2 same logic as 4+4+3+3+1: can't have two 4s

    4+3+3+3+2 can't have three repeat digits, see 5+3+3+3+1

    3+3+3+3+3 repeats across the row and columns, impossible

    So, the only combination we haven't ruled out is 5+4+3+2+1, without any repeat digits. Note we didn't even need to solve 14+ and 13+ areas, nor the corners A5 and E1.
