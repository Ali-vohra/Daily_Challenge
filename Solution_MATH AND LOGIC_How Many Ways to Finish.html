How Many Ways to Finish?

How much information do you need to put three things in order, like the finish order of a three-person race? It turns out, not much at all.

Suppose that, in a race among Amelia, Becca, and Carlos, we only know one fact: Becca finished before Carlos. What are the possible orders for the entire race? Keep reading to see some reasoning we can use on order puzzles like these, or jump ahead to today's challenge for a trickier problem.

We can often make order problems like these easier by focusing on a single unknown factor and then testing different cases for that unknown. For example, in this problem, we can focus on "where did Amelia finish?" and then test different cases to see if they conflict with any facts. def check(in_data):
    """
    in_data['A']: position of Amelia
    in_data['B']: position of Becca
    in_data['C']: position of Carlos
    in_data['D']: position of Dara
    checks if all are as described in:
        https://brilliant.org/daily-problems/order-puzzle-3/
    """
    res = True
    if 'A' in in_data and 'C' in in_data and not in_data['A'] < in_data['B']:
        res = False
    if res and 'D' in in_data and 'B' in in_data and not in_data['D'] < in_data['B']:
        res = False
    if res and 'B' in in_data and 'C' in in_data and not in_data['B'] < in_data['C']:
        res = False
    if res and 'D' in in_data and not in_data['D'] > 1:
        res = False
    return res

def get_all_solutions(key_list, check_fun):
    """
    find the correct order of key_list defined by check_fun
    """
    pos_list = list(range(1, len(key_list)+1))
    sol_set = set()
    def inner(cur_sol_data, left_pos_list):
        if check_fun(cur_sol_data):
            if len(left_pos_list) > 0:
                cur_num = len(cur_sol_data)
                cur_key = key_list[cur_num]
                for (tmp_pos_num, tmp_pos) in enumerate(left_pos_list):
                    tmp_sol_data = cur_sol_data.copy()
                    tmp_sol_data[cur_key] = tmp_pos
                    tmp_left_pos_list = left_pos_list.copy()
                    del tmp_left_pos_list[tmp_pos_num]
                    inner(tmp_sol_data, tmp_left_pos_list)
            else:
                sol_str = ' '.join([f'{_}: {cur_sol_data[_]}' for _ in cur_sol_data])
                sol_set.add(sol_str)
    inner({}, pos_list)
    return sol_set

SOL_SET = get_all_solutions(['A', 'B', 'C', 'D'], check)
for _ in SOL_SET:
    print(_)

print(f"Number of solutions: {len(SOL_SET)}")
