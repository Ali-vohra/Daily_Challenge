Neuron Logic

The heart of a computer is Boolean logic. All of a computer's complexity ultimately rests on logical operations like AND,OR,NOT,\mathbf{AND}, \mathbf{OR}, \mathbf{NOT},AND,OR,NOT, and NAND\mathbf{NAND}NAND acting mechanistically on the information contained in bits.

Problems that break into a sequence of logical steps are ripe for solving with a computer. But what if the logic of a particular problem is beyond your reach?

Enter the artificial neural network.

An artificial neural network (ANN) is a system that can learn the logic of complex patterns in data. Its basic unit is the artificial neuron, inspired by simplified mathematical models of the neurons in animal brains.

Just as biological neurons send and receive electrical impulses to other neurons and change their own state based on the pulses they receive, the artificial neuron can be active (■)(\blacksquare)(■) or inactive (□)(\square)(□) based on the information it receives from its inputs.

The inputs with green edges drive the neuron toward its active state (■).(\blacksquare).(■). The pink input has the opposite effect, inhibiting the neuron when it's on.

The neuron's output state is determined by summing the inputs: each green edge contributes +1+1+1 and pink one contributes −1.-1.−1. (Only the inputs that are on are counted; inputs switched off have no effect.)

When the sum of the inputs equals or exceeds a particular value, called its bias, the neuron flips from □\square□ to ■.\blacksquare.■. The higher the bias, the greater the input needed to flip the neuron.

The leftmost input alone isn't enough to flip the neuron to ■,\blacksquare,■, but if you set both green inputs to on, their combined effect is. Likewise, the pink neuron lowers the total input, and it flips the output neuron back to its inactive state (□).(\square).(□).

With this rule, a neuron can be configured to perform basic logical operations (or "gates") on the inputs. For example, the neuron below mimics the logical AND\mathbf{AND}AND gate.

I have provided a more general and logical solution to answer which are the only two inputs rather than trial and error method. It is long so don't review before reading the whole thing.

First of all, we have to choose two inputs out of the four inputs so let's name the green inputs G1, G2 and the pink inputs as P1, P2.

So the possible pairs of inputs are :

  • G1 ⇒ G2
  • G1 ⇒ P1
  • G1 ⇒ P2
  • G2 ⇒ P1
  • G2 ⇒ P2
  • P1 ⇒ P2

Note: Input A ⇒ Input B is same as Input B ⇒ Input A due to symmetry. For ex, G1 ⇒ P2 is the same as P2 ⇒ G1. Hence above we only have 6 possible pairs

The given things are: A green input G can have the following values: 1 for ON and 0 for OFF

A pink input can have the following values: -1 for ON and 0 for OFF

Also, we are allowed to only select 2 inputs and thus the other 2 inputs would be OFF i.e their value to be 0 so that they don't alter the result. For ex, If the pair G1 ⇒ P1 is selected then the inputs G2 and P2 would be OFF.

Now in a NAND gate, the output would only be OFF if both the inputs are ON.

So for a selected pair, the sum of both the inputs should be always lesser than the bias ( as when the sum of both inputs is greater than or equal to the bias the output is ON, so for the output to be OFF we need the sum of both inputs should be less than the bias) and thus any other value i.e if one input is OFF and the other is ON or both are OFF should be greater than or equal to the bias ). The bias has to be just greater than the sum and thus it has to be only 1 unit greater than the sum of the inputs. ( as 1 unit is the smallest we can move while arranging the bias scale)

Therefore, the formula for Bias = Sum of inputs + 1

So a trick here is that the pairs :

  • G1 ⇒ P1
  • G1 ⇒ P2
  • G2 ⇒ P1
  • G2 ⇒ P2

are identical as they all are a pair where it is one Green input and one Pink input so they all yield the same values so disproving any one pair disproves all the 4 pairs.

So let's consider the pair G1 ⇒ P1 where if both inputs are ON we get the sum as Green input + Pink input = 1-1 = 0. Therefore Bias = 0 + 1 = 1. However, suppose only P1 is ON then the sum is 0 + (-1) = -1 which is also lesser than the bias which cannot be for a NAND gate as stated above. So all the four pairs are wrong.

Now the only remaining pairs are:

  • G1 ⇒ G2
  • P1 ⇒ P2

Lets now consider the pair G1 ⇒ G2 where if both inputs are ON we get the sum as 1+1=2 so the bias should be 2+1=3 but if any one input is ON then the value is 0+1=1 which is also lesser than the bias and thus even the pair G1 ⇒ G2 is wrong.

Now the only pair remaining is P1 ⇒ P2 where if both the inputs are ON then the sum is -1 + (-1) = -2 and the bias is then -2+1=-1 and also if any one input is on the value is -1+0=-1 which is equal to the bias and if both are off the value is 0+0 = 0 which is greater than the bias and thus the pair P1 ⇒ P2 is right as it follows the conditions stated above.

So the only two inputs are P1 and P2.

Using the Distributive Property on Boolean Algebra you get:

NOT ( A AND B ) = ( NOT A ) ( NOT AND ) ( NOT B ) = ( NOT A ) OR ( NOT B )

In the discussion for positive inputs, a bias level of +1 corresponded to the OR function and a bias level of +2 corresponded to the AND function.

The result of the equation above says that we are using negative (pink) inputs and that we want to use an OR function on the negative inputs. A bias of -1 corresponds to OR, in the negative sense, and a bias of -2 would correspond to AND.

Q.E.D. Use the pink inputs (i.e. 3 and 4) ... and set the bias to -1 to get an OR relationship for these inputs resulting in a NAND gate.

Note that you can get a NOR gate using pink inputs and a bias of -2, since NOT ( A OR B ) = ( NOT A ) AND ( NOT B ).
