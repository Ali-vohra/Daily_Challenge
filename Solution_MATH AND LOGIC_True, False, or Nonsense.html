True, False, or Nonsense?

Of the three statements below, how many can be true at the same time?

Keep reading to see how we can reason our way to an answer, or jump ahead to today's challenge for a trickier conundrum.

In situations like this, it's easy to get confused and lose track of which statements are true and which aren't. Let's make a truth table to organize the possible outcomes. We'll start by assuming that statement A is true:

ABCT \Large \def\arraystretch{1.5} \begin{array}{ccc} \text{A} & \text{B} & \text{C} \\ \hline \text{T} & & \\ \end{array} AT​B​C​​

Since statement A is "statement B is false" and we're assuming it's true, statement B must be false. The negation of statement B is "statement A is true," so B being false is consistent with our assumption that statement A is true.

If we assume statement C is true, that indeed gives us two logically consistent statements: A and C. If we assume statement C is false, then statement A is the only true statement. So, when statement A is true, we can have either one or two true statements:

ABCTFTTFF \Large \def\arraystretch{1.5} \begin{array}{ccc} \text{A} & \text{B} & \text{C} \\ \hline \text{T} & \text{F} & \text{T}\\ \text{T} & \text{F} & \text{F}\\ \end{array} ATT​BFF​CTF​​

What if statement A were false? The negation of statement A is "statement B is true." Since we're assuming statement A is false, statement B is indeed true. Along with statement C, this gives us two more ways we can have one or two true statements:

ABCTFTTFFFTTFTF \Large \def\arraystretch{1.5} \begin{array}{ccc} \text{A} & \text{B} & \text{C} \\ \hline \text{T} & \text{F} & \text{T}\\ \text{T} & \text{F} & \text{F}\\ \text{F} & \text{T} & \text{T}\\ \text{F} & \text{T} & \text{F}\\ \end{array} ATTFF​BFFTT​CTFTF​​

It's not possible for all three statements to be true. One reason is that if statement C is true, then only two statements can be true, which would be a contradiction. Another reason is that statements A and B are not logically consistent — they can't be true at the same time.

Let's change our statements a bit. How many statements can be true at the same time now?

Let's start as we did with the last example, and assume statement A is true, which means statement B is false. The negation of statement B is "statement A is false." This doesn't make sense since we've assumed statement A is true.

So, let's say that statement A is false. That means statement B must be true... but if statement B is true, then so is statement A. This gives us another contradiction. We can't interpret statement A or B as either true or false. If we try to do so, we end up with a contradiction no matter what. When considered together, they're pretty much nonsense.

If we assume statement C is true, then we indeed have only one true statement since it's referring to itself. Assuming C is false leaves us with no true statements:

We see that since Alice's, Mad Hatter's and Cheshire Cat's statements exclude one another, at least two of them have to be false. Hence we have at most 3 true statements.

  • If exactly 3 statements are true, then Alice's statement is true while Mad Hatter's and Cheshire Cat's statements are false. This implies that March Hare's and Dormouse's statements are true. However, March Hare contradicts Alice so this cannot be the case.

  • If exactly 2 statements are true, then Mad Hatter's statement is true while Alice's and Cheshire Cat's statements are false. However, this means that March Hare's and Dormouse's statements are both true, yielding a total of three true statements, a contradiction.

  • If exactly 1 statement is true, then Cheshire Cat's statement is true, implying all the other statements should be false. However, March Hare's statement being false means that Alice's statement should be true as well, a contradiction.

So none of the statements can be true without contradictions. Furthermore, even if we assumed none of the statements were true, then in particular Alice's statement would be false. This would mean, however, that March Hare's statement is true, yielding a contradiction once again.

from itertools import permutations

truth_values = []
for _ in range(5):
    truth_values.append(True)
    truth_values.append(False)

def is_it_possible(Alice, March, Mad, Dormouse, Cheshire):
    number_of_truth_statements = Alice + March + Mad + Dormouse + Cheshire

    if (number_of_truth_statements == 3) and (Alice == False):
        return False

    if (number_of_truth_statements == 2) and (Mad == False):
        return False

    if (number_of_truth_statements == 1) and (Cheshire == False):
        return False

    # Alice and March cannot share the same truth values
    if (Alice + March == 0) or (Alice + March == 2):
        return False

    # Cheshire and Dormouse cannot share the same truth values
    if (Cheshire + Dormouse == 0) or (Cheshire + Dormouse == 2):
        return False

    # At most one of Alice, Mad and Cheshire's statements must be true
    if (Alice + Mad + Cheshire > 1):
        return False

    return True

solution_found = 0
max_truth_statements_possible = 0

for all_possible_results in list(permutations(truth_values, 5)):
    if (is_it_possible(all_possible_results[0], all_possible_results[1], all_possible_results[2], all_possible_results[3], all_possible_results[4]) == True):
        solution_found += 1
        if sum(all_possible_results) > max_truth_statements_possible:
            max_truth_statements_possible = sum(all_possible_results)

print("Solutions found:\t" , solution_found)

if solution_found > 0:
    print(max_truth_statements_possible)

I found this to be an unexpectedly simple logic puzzle resulting from the three contradictory declarative statements - one, two, or three statements being true.

Starting with Alice (3T), the statements of both the Mad Hatter and Cheshire Cat must be false. The March Hare’s statement must also be false because Alice is true. Therefore, there cannot be exactly three true statements with Alice’s assertion being true.

Going to the Mad Hatter (2T), the statements of both Alice and the Cheshire Cat must be false. But that makes the statements of the March Hare and Dormouse true, thus violating the Mad Hatter’s assertion.

If the Cheshire Cat speaks the truth (1T), then the statements of both Alice and the Mad Hatter must be false. But that means that the March Hare’s statement is true. Given at least two true statements, the Cheshire Cat’s assertion is violated.

If we want to be thorough, then we must test the case in which the statements of the Chsire Cat, Mad Hatter, and Alice are all false. In that case, there cannot be greater than two true statements, so the high side (>3) is addressed. However, the statements of the March Hare and the Dormouse are both true in this case, so there is no possibility of all statements being false.

The only acceptable answer is that none of the statements can be true without contradictions.
