# last digit in grid indicates the non-used tile in ascending order

def eval(n, grid, used, solutions):
    if n == 9:
        if grid.count(1):
           solutions.add(tuple(sorted(grid[:2]) + sorted(grid[2:4]) + sorted(grid[4:6]) + sorted(grid[6:8]) + [grid[-1]]))
        return
    for tile in range(1, 10):
        if tile in used:
            continue
        grid[n] = tile
        if n == 3 and not grid[2:4].count(min(grid[:2])**2):
            continue
        if n == 5 and not grid[2] * grid[3] == int(str(grid[4]) + str(grid[5])):
            continue
        if n == 7 and not grid[4] * grid[5] == int(str(grid[6]) + str(grid[7])):
            continue
        used.add(tile)
        eval(n+1, grid, used, solutions)
        used.remove(tile)
    return solutions

Farrah can't have the 111-tile, because if she did, then Kelly would need a 111-tile, too, since 12=11^2 = 112=1. Kelly and Letoya can't have the 111-tile, either, because if either of them did, then the product of their tiles would be one digit, which would mean Letoya or Michelle would have one tile instead of the required two. Therefore, Michelle must have the 111-tile.

I think I may be the 3rd person to post an explanation. Here it is (Purely from trial and error and logic.): Kelly states that one of her tiles is the SQUARE of Farrah's smallest tile. Therefore, the smallest tile of Farrah has to be either 2 or 3 (as 4^2 = 16 which is not a single digit.) Let's start by assuming that it is 2.

CASE 1 (One of Farrah's Tiles is 2): By using 2, we get that one of Farrah's tiles has to be 2, and one of Kelly's tiles has to be 4. Letoya states that her tiles are the digits of Kelly's product. Therefore, the other tile of Kelly has to be less greater than or equal to 3 (as it results in a single digit number otherwise.) Using 3, we get 43 = 12. However, 2 was already used so 3 cannot be a tile of Kelly's. Using 5, we get 45 = 20. However, as there is no 0, this is also not possible. Using 6, we get 46 = 24 which is not possible (as 2 and 4 have been used.) Using 7, we get 28 which is again not possible (as 2 has been used). Using 8, we get 32, which is again not possible as 2 has been used. Using 9 (finally) we get 36. Both of these numbers have not been used, so we can assume Kelly's other tile to be 9. Letoya's tiles hence are 3 and 6. Michelle states that his tiles are the product of the tiles of Kelly. Therefore, it is 36 = 18. (1,8 are the tiles). The only tiles left are 5 and 7. Therefore, two of the correct options are 5 and 7.

CASE 2(One of Farrah's Tiles is 3): By using 3, we get that one of Farrah's tiles is 3. Following the same logic mentioned above, we get that one of Kelly's tiles has to be 9. By taking the other tile of Kelly to be 2, we get 18. However, we will want to avoid forming a product with the digit 1 as it can lead to 18 = 08, and as 0 is unavailable, it is not a possible solution. Using 4, we get 36, which (again) is not possible.
