Circles on the Scale

Given the two balance scales, we might ask how much the green triangle weighs:

Or perhaps, we might want to know the value of ttt in the equations s+t=32s+t=5. \begin{aligned} s + t &= 3 \\ 2s + t &= 5. \end{aligned}s+t2s+t​=3=5.​ In either case, we can use the same approach: the elimination method.

Elimination depends on this fact:

We can always keep a scale in balance by removing (or adding) equivalent quantities on both sides.

What's happening here? We know from the first balance that a square and a triangle are equivalent to 333 (they are in balance and must therefore weigh the same). Therefore, on the second balance, we can remove a square and a triangle from the left and 333 from the right, and the scale will still be in balance.

Note that although what we removed on the right looks different from what we removed on the left, we know that they are the equivalent because the first scale showed us.

From here, we can solve directly: the blue square equals 2,2,2, so the green triangle must equal 1.1.1.

Let's look at the same approach but this time with an algebraic representation: s+t=32s+t=5. \begin{aligned} s + t &= 3 \\ 2s + t &= 5. \end{aligned}s+t2s+t​=3=5.​ Subtracting the first equation from the second, we get 2s+t=5−(s+t)=−(3)⇒s=2. \begin{aligned} 2s + t &= 5 \\ - (s + t) &= -(3) \\ \Rightarrow s &= 2. \end{aligned} 2s+t−(s+t)⇒s​=5=−(3)=2.​ Because s=2s=2s=2, we can conclude that t=1t=1t=1. By combining these equations, we found the solution very quickly.

Although the problem below looks cumbersome, if you approach it as an elimination problem — where combining the equations together lets you simplify — there's a straightforward path to the solution.

    It is the straightforward that the 3rd balance shows that 2C=1T+1S (by removing a T from each side).

    So 4C=2T+2S is a valid answer (/balance).

    This observation, and assuming that none of the weights are ≤0, eliminates 2S+1T, 2S+3T, and 3S+2T as possible solutions.

    But what about 3S+1T? I.e., might S=T, making this a valid solution? The 3rd balance in and of itself does not exempt this as a possible solution.

    The first balance - 2T+1C = 2S+2C - does though, as if T=S, then C=0, which is not allowed. (If C=0 is allowed, then this implies that T = 0 and S = 0, and all of the offered answers would be correct!!!).
