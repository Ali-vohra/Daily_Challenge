How Many Ways to Fill the Grid?

Grid Fill puzzles like the one below are similar to Sudoku puzzles in that they ask you to place numbers in squares in a certain way, but the similarities end there.

If you're familiar with how these work, jump straight to today's problem. If you want to look through some worked examples, though, keep reading.

One of the differences between these puzzles and Sudoku is that a Grid Fill solution can have a number appearing more than once in the same row or column. The goal is to drag number tiles from the bank to squares on the grid so the rows and columns sum to the indicated values.

For example, a solution to the puzzle below will have numbers in the top row with a sum of 3. Note that the middle square of that row is gray and will not accept a number tile, so its effective value is zero. Because each row and column has an even sum, any solution must have a 3 and a 1 in each column to ensure that the odd numbers add together to create an even sum. Because this puzzle does not allow us to place any additional 1s, all solutions force us to pair each 3 with a 1 for every row and column with a 1 in it. Because the second column has no odd numbers pre-placed, all solutions must have the full column be nothing but even numbers, resulting in it being only 2s. This leaves us with only 6 potential positions in which to place 3s, 2 per row and remaining column due to the placement of the 1s. This means that there are only 2 possible solutions for a single row. Solving for any single row results in the elimination of potential solutions for the remaining rows, due to the fact that any solution cannot have duplicate 3s in any row or column. As a result, the solution for a single row extends to a solution for the entire puzzle. Therefore, because there are only 2 solutions for a single row, there are only 2 solutions to the entire puzzle.
