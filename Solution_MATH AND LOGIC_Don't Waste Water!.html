Don't Waste Water!

Two containers: one holds 333 liters and the other holds 5.5.5.

The challenge: measure a specific quantity of liquid using only a sequence of actions chosen from the legal moves below.

We can use these moves to measure amounts of water that are different from the exact measurements on the bottles. For example, if we fill the 555-liter container and pour it into the 333-liter container until it is full, we can be certain that we have 222 liters remaining in the larger bottle.

We can also combine multiple steps to find even more measurements. For example, we can measure out 111 liter by doing the following:

  1. Fill the 333-liter bottle.
  2. Empty the 333-liter bottle into the 555-liter bottle.
  3. Refill the 333-liter bottle.
  4. Pour the 333-liter bottle into the 555-liter bottle until the larger bottle is full.

This leaves us with exactly 111 liter left in the smaller bottle.

Today's Challenge

You have one container that holds 444 liters and another container that holds 777 liters.

These are the only moves possible:

  • Completely fill a container from the tap.
  • Completely empty a container onto the ground.
  • Pour a container into another until one is completely full or the other is completely empty,

Using only the moves listed above, what is the smallest amount of water you need to pour onto the ground in order to measure 222 liters in either one of the containers?

444 liters 777 liters 888 liters 121212 liters 141414 liters It is not possible to measure out 222 liters.

import networkx as x
# each function generates nodes and costs
def pour_a_to_b(a, b, max_b):
    final = min(max_b, a + b)
    poured = final - b
    yield (a - poured, b + poured)

def pours(a, b, maxes):
    l, r = pour_a_to_b(a, b, maxes[1])
    yield (l, r, 0)
    r, l = pour_a_to_b(b, a, maxes[0])
    yield (l, r, 0)

def dump(a, b, maxes):
    yield from [(0, b, a), (a, 0, b)]

def fill(a, b, maxes):
    yield from [(a, maxes[1], 0), (maxes[0], b, 0)]

# define the buckets
maxes = [7, 4]

# make the graph
to_open = [(0, 0)]
G = nx.DiGraph()
for a, b in to_open:
    for f in [pours, dump, fill]:
        for a2, b2, wt in f(a, b, maxes):
            if (a2, b2) not in to_open:
                to_open.append((a2, b2))
            G.add_edge((a, b), (a2, b2), weight=wt)

# find the the lowest cost paths
for t in [node for node in G if 2 in node]:
    c = nx.shortest_path_length(G, (0,0), t, weight='weight')
    print(t, c)

>>> (2, 0) 12
>>> (7, 2) 7

To keep things short, I will not include here how we find that we need to waste 7L to get 2L. Instead I will just show why 4L can’t be an option. Proof by contradiction.

Assume we can. Then there are two paths to waste 4L. Either we waste it by the path where we pour 4L from the 4L container or the path we pour 4L from the 7L container. There are two possible paths by pouring 4L from the 4L container. Either we fill it and pour directly or fill it through the 7L container. But in both cases we can not get 2L. Contradiction.

The other path was to waste it by pouring 4L from the 7L container. There is only one way: Filling the 4L container and pour this to the 7L container and then empty the 7L container. But this is certainly not giving us 2L either. Contradiction.

There are two points that I want to mention, just for completeness. I will use today's example of 4 and 7 liter containers but the point is true in a more generalized sense. Mentioning these two points in no way diminishes your work, or the work of the many others who participated in this challenge.

Using only 4 and 7 liter containers, you can MEASURE any non-negative amount of water, i.e. 0 to infinity (assuming you have the stamina ...). Of course, with only these two containers, you can only HOLD, at most, 11 liters.

In today's problem, the goal was to measure and hold 2 liters of water in either container, constrained by the consideration of wasting the least amount of water. There are an infinite number of ways to measure and hold 2 liters of water using only these two containers. All of the solutions are periodic. There are two solutions that are the FIRST occurrences of these periodic solutions, and one of these two solutions will waste the least amount of water.
