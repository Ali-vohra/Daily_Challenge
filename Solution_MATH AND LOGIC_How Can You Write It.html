How Can You Write It?

If we want to write a number as a difference of two squares, it's helpful to think about the difference of squares identity and how it works.

The difference of squares identity is this:

a2−b2=(a+b)(a−b). a^2 - b^2 = (a+b)(a-b).a2−b2=(a+b)(a−b).

Because it's an identity, these two sides of the equation will always be equal, even though they look quite different.

To see how this can be helpful, let's try to write 111111111 as a difference of squares. We want a2−b2=111a^2-b^2=111a2−b2=111, but there are far too many numbers to choose from to do this by guess and check. This is true for a small number like 111,111,111, but the problem only gets harder with larger numbers.

However, we can look at the right-hand side of the identity above and note that it's a multiplication problem: (a+b)(a−b).(a+b)(a-b).(a+b)(a−b). If we could write 111111111 as a product of two numbers, we might be able to use that information to find aaa and bbb.

In this case, we can find that 111=37×3111=37 \times 3111=37×3. Using the right-hand side of the equation and noting that 111=37×3=(a+b)(a−b)111=37 \times 3 = (a+b)(a-b)111=37×3=(a+b)(a−b), we have

a+b=37a−b=3. \begin{aligned} a+b &= 37 \\ a-b &=3. \end{aligned} a+ba−b​=37=3.​

The only solution to this system of equations is a=20,b=17a=20, b=17a=20,b=17, which gives us two numbers that should satisfy a2−b2=111.a^2-b^2 =111.a2−b2=111. Let's check:

a2−b2=202−172=400−289=111. \begin{aligned} a^2 - b^2 &= 20^2 - 17^2 \\ &= 400 - 289 \\ &= 111. \end{aligned}a2−b2​=202−172=400−289=111.​

Is 202−17220^2-17^2202−172 is the only way to express 111111111 as a difference of two squares? Although 333 and 373737 are prime factors of 111,111,111, there is another way to obtain 111111111 through multiplication: 111×1.111\times1.111×1. Using this factorization and substituting as we did above, we can find another way to express 111111111 as a difference of squares: 562−552=111.56^2-55^2=111.562−552=111.

What about the number of our new year? Can we reason the same way about 2020?2020?2020?
