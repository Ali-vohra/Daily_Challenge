Can You Find Them All?

When a problem presents symmetrical cases or shapes, we can use that symmetry to help organize our work and ensure accuracy.

For example, how many triangles are there in the following figure?

We can use casework and symmetry to organize a list of triangles to make sure we account for all possibilities.

Looking carefully, we find there are a total of 9 small triangles, which we can see separately.

It is also possible to make a medium-sized triangle by combining 4 small ones, as shown below.

By symmetry, we see three of these triangles, each in one corner of the larger overall shape.

Finally, the overall figure itself is a large triangle, so the total number of triangles is 9+3+1=13.9 + 3 + 1 = 13.9+3+1=13.

We can apply the same kind of approach to problems like the one below. Be careful to consider all the cases.

"""
solution of:
    https://brilliant.org/daily-problems/center-square/
"""
import itertools

def check(square_pos, square_size, grid_width, grid_height, orange_pos):
    """
    checks if a square with a corner at square_pos and size_square_size
    fits the grid and contains the orange cell
    """
    res = True
    if square_pos[0]+square_size > grid_width:
        res = False
    elif square_pos[1]+square_size > grid_height:
        res = False
    elif not (square_pos[0] <= orange_pos[0] < square_pos[0]+square_size and
              square_pos[1] <= orange_pos[1] < square_pos[1]+square_size):
        res = False
    return res

GRID_SIZE = 5
ORANGE_POS = (2, 2)
ALL_DATA_GEN = itertools.product(itertools.product(range(GRID_SIZE), repeat=2),
                                 range(1, GRID_SIZE+1))

SUM_VAL = sum((1 for (cur_pos, cur_s) in ALL_DATA_GEN
               if check(cur_pos, cur_s, GRID_SIZE, GRID_SIZE, ORANGE_POS)))
print(f"Number of squares containing the orange one: {SUM_VAL}")
