The Secret of the Resistor Maze

We are surrounded by all sorts of electric devices: some are simple like a light bulb, and some are very complicated like a power grid or a computer. Inside these devices, electricity flows around circuits, carrying energy precisely where it's needed.

Electric current is the coordinated motion of an unimaginable number of electrons, but the rules for how much current passes through a device are surprisingly straightforward.

Electric currents — flows of electric charge — are driven by differences in voltage. Current flows from positions of high voltage to positions of low voltage — similar to how water flows from high elevations to low elevations. In fact, voltage difference along a path in the circuit is the amount of potential energy per charge lost or gained.

Power sources like batteries are sources of voltage; they add potential energy to the moving charges. Resistors, like light bulbs for example, cause the voltage to drop quickly. To keep a steady current, the charges' potential energy splits up according to the circuit's resistance. Along wires, where resistance is low, the voltage barely varies at all. Each item in the circuit produces a change in the voltage, which we usually write ΔV\Delta VΔV.

Whether you're looking at a simple circuit or a complicated nest of wires and bulbs with multiple batteries, there are two rules we can follow to piece together these ΔV\Delta VΔV's and figure out how much current is supplied.

Voltage rule: The voltage must be consistent at any position, whichever way we calculate it. For example, if there is a circuit with two branches that come back together, the voltage across each branch must be the same.

Current rule: In a steady state, charge cannot build up anywhere in the circuit. This means that if we look at any point in the circuit, the current flowing out must be the same as the current flowing in.

Armed with these rules, we just need some components to slot into our circuits. These components are the building blocks we can use to build any circuit, so engineers represent them with standard symbols. Two of the most important are batteries and resistors:

  • A battery (left) uses chemical energy to add potential energy to the charges flowing through it, maintaining a fixed ΔV\Delta VΔV between its ends (regardless of the current flowing through it).
  • A resistor (right) obeys Ohm's law, which says the voltage drop ΔV\Delta VΔV across a resistor is proportional to the current III flowing through it: ΔV=−IR\Delta V = - I RΔV=−IR. The constant RRR is called the resistance of the resistor.

How can we apply these rules? As an example, let's imagine we power a light bulb (((a resistor with resistance Rbulb)R_\text{bulb})Rbulb​) with two batteries that each provide a potential gain of ΔV=+V\Delta V=+VΔV=+V. The batteries have internal resistance, which we represent with an extra resistor with resistance RinternalR_\text{internal}Rinternal​ connected in line with each ideal battery.

According to the current law, the current is the same all the way around this loop. We can add up the ΔV\Delta VΔV's, starting at some arbitrary point P. When we get back to P, we should have the same voltage, meaning the sum of ΔV\Delta VΔV's must be zero. This gives us V−IRinternal+V−IRinternal−IRbulb=0.V-IR_\text{internal}+V-IR_\text{internal}-IR_\text{bulb}=0.V−IRinternal​+V−IRinternal​−IRbulb​=0.We can rearrange this to get the total current around the circuit: I=2V2Rinternal+Rbulb.I=\frac{2V}{2R_\text{internal}+R_\text{bulb}}.I=2Rinternal​+Rbulb​2V​.

To find the current through the bulb, we need to find the voltage across it.

If we tried to find the voltage drop across each resistor, we'd first have to solve an overwhelmingly complicated system of simultaneous equations in order to find the current through each resistor. On the other hand, the voltage gain across a battery is always the same, regardless of current.

This means that if we can find a route through the maze which contains only batteries, and no resistors, we don't have to calculate any currents at all!

There is exactly one route through the maze which has only batteries:

Then, we can count that this route passes through 8 batteries, so the voltage between the two sides of the maze is 8 V\SI{8}{\volt}8 V. The voltage law tells us that if we calculated the voltage any other way, we would get the same answer.

Ohm's law then says that if there is a voltage of 8 V\SI{8}{\volt}8 V across a 1 Ω\SI{1}{\ohm}1 Ω resistor, the current through the resistor is I=VR=8 V1 Ω=8 AI=\frac{V}{R}=\frac{\SI{8}{\volt}}{\SI{1}{\ohm}}=\SI{8}{\ampere}I=RV​=1 Ω8 V​=8 A

I actually solved the system of 17 linear equations. It wasn't that hard.

Assign variables to the clockwise current in each cell as follows:

Then write an equation for the voltage deltas around each cell (which must add to zero) and solve the corresponding variable in terms of remaining variables:

-3a+c=0 => a=c/3

-b+e=0 => b=e

-2c-1+a=0 => -2c-1+c/3=0 => 5c/3=-1 => c=-3/5

-d-1=0 => d=-1

-2e+1-1+h+b=0 => -2e+h+e=0 => e=h

-f+1+i=0 => f=i+1

-2g-k-1=0 => g=-k/2-1/2

-2h+e+r+2=0 => -2h+h+r+2=0 => h=r+2

-2i+f+r+1=0 => -2i+i+1+r+1=0 => i=r+2

-2j+m-1=0 => j=m/2-1/2

-k-2=0 => k=-2

-3r+i+n+h+1=0 => -3r+r+2+n+r+2+1=0 => r=n+5

-2m+j+o-1=0 => -2m+m/2-1/2+o-1=0 => 3m/2=o-3/2 => m=2o/3-1

-2n+r+q+2=0 => -2n+n+5+q+2=0 => n=q+7

-o+m-1=0 => o=m-1

-p+q-1=0 => p=q-1

-3q+n+p+2=0 => -3q+q+7+q-1+2=0 => q=8

The last variable, q ,is the current through the bulb, 8.
