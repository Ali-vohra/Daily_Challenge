Differences of One

The goal of a Calcdoku puzzle is to fill the squares with a single digit each with two restrictions:

  • Each of the digits is used exactly once in each row and column.
  • Every group has to create the stated value by using the operator shown. (The numbers can be used in any order.)

solution of:
    https://brilliant.org/daily-problems/calcdoku-12/
"""

import itertools
def check_conditions(in_data):
    """
    Verifies, if in_data (storing only the first n-rows) satisfies
    the "arithmetic" conditions of the puzzle
    https://brilliant.org/daily-problems/calcdoku-12/
    (for which only the first n-rows are needed).
    If the input data has the size n, it assumes,
    that the result of this function for the first n-1 "rows" was True.
    """
    res = True
    if len(in_data) == 1:
        if not (in_data[0][0]/in_data[0][1] != 3 or in_data[0][1]/in_data[0][0] != 3):
            res = False
    elif len(in_data) == 2:
        if abs(in_data[0][2]-in_data[1][2]) != 1:
            res = False
    elif len(in_data) == 3:
        if abs(in_data[1][0]-in_data[2][0]) != 1:
            res = False
        elif abs(in_data[2][1]-in_data[2][2]) != 1:
            res = False
    return res

def gen_solution(in_size, in_check_arithmetic_cond):
    """
    generates all Latin squares of the order in_size, which
    satisfy the additional conditions described in in_check_arithmetic_cond
    """
    def are_all_different(in_list):
        """
        checks if all of the entries in in_list are different
        """
        return len(set(in_list)) == len(in_list)
    def add_row(cur_row_data):
        if len(cur_row_data) == in_size:
            yield tuple(cur_row_data)
        else:
            for tmp_row in itertools.permutations(range(1, in_size+1)):
                tmp_row_data = cur_row_data.copy()
                tmp_row_data.append(tmp_row)
                for col_num in range(in_size):
                    cur_col = [tmp_row_data[_][col_num] for _ in range(len(tmp_row_data))]
                    if not are_all_different(cur_col):
                        break
                else:
                    if in_check_arithmetic_cond(tmp_row_data):
                        for _ in add_row(tmp_row_data):
                            yield _
    for _ in add_row([]):
        yield _

def sol_to_str(sol_data):
    """
    returns a string representation of a solutions
    """
    return '\n'.join(' '.join(str(_) for _ in cur_row) for cur_row in sol_data)

NUMBER_OF_SOLUTIONS = 0
for cur_sol in gen_solution(3, check_conditions):
    NUMBER_OF_SOLUTIONS += 1
    print(f"solution number {NUMBER_OF_SOLUTIONS}:\n{sol_to_str(cur_sol)}")
print(f"Number of solutions: {NUMBER_OF_SOLUTIONS}")

    If 3 must be obtained by division, then the two numbers that go in that region must be 1 & 3 (in any order), implying that the number to go in the top right corner is 2. By fixing the 2 in that position, we can proceed to manipulate the numbers to satisfy the condition for each region. Note that there are two possible ways to satisfy the condition for each region (including the centre - 1 or 3), but changing the order of the numbers in any one region changes the arrangement in every other region (whether this means changing one of the numbers or rearranging the numbers as in the case of the 3 region). Therefore, there are only two possible solutions (or states) for the grid of numbers.

    2 has to go into the upper right square since the 3-divide group (upper left) must have 1 and 3 in some order. Now the bottom right 1-minus group must contain 1 and 2 or 2 and 3, so there must be a 2 in that group. Since there's a 2 in the upper right square, the two in the bottom group must go in the bottom middle square. The 3rd 2 must then go in the left middle square.

    Now it's easy to place all the 1's and 3's to complete the puzzle. Start with 3, 1 (in that order) in the upper left, and the rest of the 3's and 1's are forced. Swap the 3 and 1 in the upper left and again the rest of the 3's and 1's are forced. Since the 2's are fixed and we've tried the only 2 ways to place the 1's and 3's, these are the only 2 solutions.

    step 0) Label each of the nine cells with a letter A through J like this: A B C D E F G H J A cell can be any number 1, 2 or 3 unless the cell value is restricted.

    math 1) In order to make a result of -1 using only the numbers 1, 2, and 3 and subtraction, in any order, you have to use the number 2, which refers to these two expressions performed in any order: 1 2 or 2 3.

    math 2) In order to make a result of 3 using division, in any order, restricts the two numbers to 1 and 3 ( referring to 3 1 ).

    step 1) cells A,B are restricted to the numbers 1 and 3. Therefore, cell C is 2.

    step 2) Together cells H,J must use the number 2 since these cells have to make the result -1 using subtraction. Also notice that cell J cannot be 2 since C is 2. Therefore, cell H is 2.

    step 3) Given that cells C and H are 2, cell D is also 2.

    At this point, the cells with value 2 are fixed, like this: A B 2 2 E F G 2 J

    math 3) As soon as the value of any one of the remaining cells is determined all of the rest of the cells are determined, in order to be consistent with the rules. For example, if A is 3, this forces B and G to be 1, which then forces J and then E to be 3 and finally F to be 1.

    Q.E.D.) cell A can have two values, either 1 or 3 and thus there are two and only two solutions!
