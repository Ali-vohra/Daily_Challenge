Random Robot Roaming

The robot starts at the center dot, and then makes four moves at random. Each move is one dot to the left or to the right.

Where is the robot most likely to end up after its four moves: on one of the inner black dots, on an outer orange dot, or on a blue dot in between? Where is the robot least likely to end up? Read ahead to find out, or skip straight to today's challenge for a more advanced problem.

There are three black dots, two blue dots, and four orange dots. If each dot had an equal chance of being the robot's final dot, then the robot would be most likely to stop on an orange dot, and least likely to stop on a blue dot. Is that the case, though?

Sometimes adding some structure can help us see what's going on more clearly. Let's number the dots, with the center dot 0:0:0:

Adding these coordinates lets us look at the problem a little differently. We'll think of moving one dot to the right as adding 111 to the robot's position, while a move to the left is subtracting 1.1.1. For example, the sequence left, right, left, left would put the robot at dot 0−1+1−1−1=−2.0 - 1 + 1 - 1 - 1 = -2.0−1+1−1−1=−2.

The sequence left, left, left, right would also leave the robot on the blue dot at −2.-2.−2. Only the number of left moves and the number of right moves matter, not the order of the moves.

How might the robot end up at dot 1?1?1? Instead of using a trial and error method, let's see if we can solve for the numbers of left and right moves. We'll call the number of left moves lll and the number of right moves r.r.r. This means lll moves to the left subtract l×1l \times 1l×1 from the robot's position, and rrr moves to the right add r×1.r \times 1.r×1. So, we need a pair of integers lll and rrr that satisfy

r−l=1, r - l = 1, r−l=1,

with the condition that r+l=4r + l = 4r+l=4 since the robot makes four moves in all. This is impossible, though. If r−l=1,r - l = 1,r−l=1, then one of the numbers is even and the other is odd. This means r+lr + lr+l will be odd, but we know that r+l=4.r + l = 4.r+l=4. There's no way for the robot to end up at dot 111 after four moves.

For rrr and lll to have an even sum, they must have the same parity. So, if the robot ends up at dot d=r−l,d = r - l,d=r−l, then ddd is even. The final dot must be an even number of dots away from the starting dot. There are only five dots the robot can land on after four moves:

We saw that the robot can only land on a dot with an even coordinate. In the challenge below, the robot can now move in four directions. The formula bar presents the formula on the center of the field. I also add the formulas for summing values for each color. The rational is this: any cell may receive the robot from any direction. But a robot in a direction might go to any of 4 different directions. So, the probability of receiving the robot from one particular direction is equal to 1/4 of the probability of having the robot there. The total probability is equal to the sum of the probabilities for each of the 4 directions each cell has. Now, we start by writing 1 in the central cell of the first field. In the second field, the central cell has, of course probability 0, and its neighbors, 1/4 each. And so on.

std::string fieldCode(const std::pair<int, int>& pos)

    return fieldCode(pos.first, pos.second);

unsigned findNumberOfColors(const std::unordered_map<std::string, unsigned>& board)
{
    unsigned curMax = 0;
    for(const auto curData : board)
    {
        if(curData.second > curMax)
        {
            curMax = curData.second;
        }
    }
    return curMax+1;
}

std::pair<int, int> makeMove(const std::pair<int, int>& inPos, const int dirCode)
{
    std::pair<int, int> resPos = inPos;
    switch(dirCode)
    {
        case 0:
            ++resPos.second;
            break;
        case 1:
            --resPos.first;
            break;
        case 2:
            --resPos.second;
            break;
        case 3:
            ++resPos.first;
            break;
        default:
            throw std::invalid_argument("Wrong value of dirCode.");
    }
    return resPos;
}

std::string toColorName(const unsigned colorCode)
{
    std::string res;
    switch(colorCode)
    {
        case 0:
            res = std::string("Green");
            break;
        case 1:
            res = std::string("Blue");
            break;
        case 2:
            res = std::string("Orange");
            break;
        default:
            throw std::invalid_argument("Wrong value of colorCode.");
    }
    return res;
}

template <typename RandomEngineType>
std::pair<int, int> randomWalk(const std::pair<int, int>& startPos, const unsigned walkLength, RandomEngineType& randomEngine)
{
    std::pair<int, int> curPos = startPos;
    std::uniform_int_distribution<> dirGenerator(0, 3);
    for(unsigned i = 0; i<walkLength; ++i)
    {
        const int curDirCode = dirGenerator(randomEngine);
        curPos = makeMove(curPos, curDirCode);
    }
    return curPos;
}

template <typename RandomEngineType>
std::vector<std::uintmax_t> singleWalkSimulation(const std::unordered_map<std::string, unsigned>& board,
                                                 const std::pair<int, int>& startPos,
                                                 const unsigned walkLength,
                                                 const std::uintmax_t localMaxIter,
                                                 const int inSeed)
{
    const unsigned numberOfColors = findNumberOfColors(board);
    std::vector<std::uintmax_t> histData(numberOfColors, 0);
    RandomEngineType randomEngine(inSeed);
    for(std::uintmax_t curIter = 0; curIter < localMaxIter; ++curIter)
    {
        const std::string resPosCode = fieldCode(randomWalk(startPos, walkLength, randomEngine));
        const unsigned resColorCode = board.at(resPosCode);
        histData[resColorCode]++;
    }
    return histData;
}

template<typename RandomEngineType>
class Worker
{
    private:
        std::unordered_map<std::string, unsigned> board;
        std::pair<int, int> startPos;
        unsigned walkLength;
        std::uintmax_t localIter;
        int seed;
        std::vector<std::uintmax_t> resultHist;
    public:
        explicit Worker(const std::unordered_map<std::string, unsigned>& _board,
                        const std::pair<int, int>& _startPos,
                        const unsigned _walkLength,
                        const std::uintmax_t _localIter,
                        const int _seed) :
        board(_board),
        startPos(_startPos),
        walkLength(_walkLength),
        localIter(_localIter),
        seed(_seed),
        resultHist()
        {}
        void operator()()
        {
            this->resultHist = singleWalkSimulation<RandomEngineType>(this->board,
                                                                      this->startPos,
                                                                      this->walkLength,
                                                                      this->localIter,
                                                                      this->seed);
        }
        std::vector<std::uintmax_t> GetResultHist() const
        {
            return this->resultHist;
        }
};

template<typename ValueType, template <typename, typename> class ContainerType >
ValueType getValueOrZero(const ContainerType<ValueType, std::allocator<ValueType> >& inData, std::size_t pos)
{
    ValueType res = ValueType(0);
    if(pos < inData.size())
    {
        res = inData[pos];
    }
    return res;
}

std::vector<std::uintmax_t> addData(const std::vector<std::uintmax_t>& histA, const std::vector<std::uintmax_t>& histB)
{
    const std::size_t resLength = std::max(histA.size(), histB.size());
    std::vector<std::uintmax_t> resHist(resLength);
    for(std::size_t i = 0; i < resLength; ++i)
    {
        resHist[i] = getValueOrZero(histA, i)+getValueOrZero(histB, i);
    }
    return resHist;
}

template <typename RandomEngineType>
std::vector<std::uintmax_t> parWalkSimulation(const std::unordered_map<std::string, unsigned>& board,
                                              const std::pair<int, int>& startPos,
                                              const unsigned walkLength,
                                              const std::uintmax_t totalIter,
                                              const unsigned numberOfThreads)
{
    if(totalIter%std::uintmax_t(numberOfThreads) != 0)
    {
        throw std::invalid_argument("totalIter must be a multiple of numberOfThreads.");
    }
    const std::uintmax_t localIter = totalIter/std::uintmax_t(numberOfThreads);
    std::vector<std::thread> threadVector;
    threadVector.reserve(numberOfThreads);
    std::vector<Worker<RandomEngineType> > workerVector;
    workerVector.reserve(numberOfThreads);
    for(unsigned i = 0; i < numberOfThreads; ++i)
    {
        workerVector.push_back(Worker<RandomEngineType>(board, startPos, walkLength, localIter, i+time(NULL)));
        threadVector.push_back(std::thread(std::ref(workerVector[i])));
    }
    for(auto & th : threadVector)
    {
        th.join();
    }

    auto sumFun = [](const std::vector<std::uintmax_t> tmpSum, const Worker<RandomEngineType>& inWorker)->std::vector<std::uintmax_t>
    {
        return addData(tmpSum, inWorker.GetResultHist());
    };
    std::vector<std::uintmax_t> restHist = std::accumulate(workerVector.begin(), workerVector.end(), std::vector<std::uintmax_t>(), sumFun);
    return restHist;
}

int main()
{
    const unsigned numberOfThreads = std::thread::hardware_concurrency();
    const std::uintmax_t totalIter = std::uintmax_t(numberOfThreads)*10000000ULL;
    auto startTime = std::chrono::steady_clock::now();
    const auto res = parWalkSimulation<std::mt19937_64>(generateBoard(), std::make_pair(0, 0), 4, totalIter, numberOfThreads);
    auto endTime = std::chrono::steady_clock::now();
    std::chrono::duration<double> runTime = endTime-startTime;
    std::cout<<"Runtime: "<<runTime.count()<<" [s]"<<std::endl;
    std::cout<<"Number of simulations: "<<totalIter<<std::endl;
    std::cout<<"endFieldColor\tnumberOfEndings\tprobability"<<std::endl;

    std::uintmax_t sum = 0;
    for(std::size_t i = 0; i<res.size(); ++i)
    {
        sum += res[i];
        std::cout<<toColorName(i)<<'\t'<<res[i]<<'\t'<<static_cast<long double>(res[i])/static_cast<long double>(totalIter)<<std::endl;
    }
    if(sum != totalIter)
    {
        throw std::overflow_error("Something went wrong. Possibly an overflow?");
    }
    return 0;
}
